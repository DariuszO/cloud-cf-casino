Sentiment Service API

GET     https://<appname>.cfapps.us10.hana.ondemand.com/sentiment/casinos
 Returns the aggregated sentiment value for each known casino.
Example: 
  [
    { "casinoid": "Las Vegas Grand Casino", "sentiment": "62.5" }, 
    { "casinoid": "Monte Carlo Casino Royal", "sentiment": "68.0" }
  ]
Response: 200 method returns successfully 
GET     https://<appname>.cfapps.us10.hana.ondemand.com/sentiment/casinos/{casinoId}
 Returns the aggregated sentiment value for the specified casino.
Example: 
  { "casinoid": "Las Vegas Grand Casino", "sentiment": "62.5" }
Response: 200 method returns successfully 
GET     https://<appname>.cfapps.us10.hana.ondemand.com/sentiment/casinos/{casinoId}/slotMachines
 Returns the aggregated sentiment value for all slot machines of the specified casino.
Example: 
  [ 
    { "casinoid": "Las Vegas Grand Casino", "slotmachineid": "Black Jack 1", "sentiment": "55.4" }, 
    { "casinoid": "Las Vegas Grand Casino", "slotmachineid": "One-Armed Bandit", "sentiment": "87.4" }
  ]
Response: 200 method returns successfully 
GET     https://<appname>.cfapps.us10.hana.ondemand.com/sentiment/casinos/{casinoId}/slotMachines/{slotMachineId}
 Returns the aggregated sentiment value for the given slot machine of the specified casino.
Example: 
  { "casinoid": "Las Vegas Grand Casino", "slotmachineid": "Black Jack 1", "sentiment": "55.4" }
Response: 200 method returns successfully 
POST     https://<appname>.cfapps.us10.hana.ondemand.com/sentiment/casinos/{casinoId}/slotMachines/{slotMachineId}/image
 Returns the aggregated sentiment value for each known casino.
Example body: 
  { "imageUrl": "<url-to-image>" }
Response: 201 new sentiment value successfully detected and persisted 
POST     https://<appname>.cfapps.us10.hana.ondemand.com/sentiment/casinos/{casinoId}/slotMachines/{slotMachineId}/dataUri
 Returns the aggregated sentiment value for each known casino.
Example body: 
  { "dataUri": "<dataUri>" }
Response: 201 new sentiment value successfully detected and persisted
